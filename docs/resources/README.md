### Global boolean "flags" implemented as functions

```bash
### cut ###
filename_only () { false; }
line_numbers  () { false; }
inverted      () { false; }

### cut ###

    # Process options
    while getopts :ilnxv opt; do
        case $opt in
            l) filename_only () { true; } ;;
            n) line_numbers  () { true; } ;;
            v) inverted      () { true; } ;;
            i) shopt -s nocasematch ;;
            x) pattern_fmt='^%s$' ;;
            ?) echo "Unknown option: $OPTARG" >&2 ;;
        esac
    done
    shift $((OPTIND - 1))

### cut ###
```
